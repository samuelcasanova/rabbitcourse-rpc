<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

You will need a RabbitMQ instance in localhost port 5672 with default username/password 'guest'

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/samuelcasanova/rabbitcourse-rpc.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Start the service
   ```sh
   npm start
   ```
